Warehouse Management Tool

A lightweight warehouse management system designed for small businesses such as bars, cafés, and restaurants.

This project is currently a **work in progress**.

---

## Overview

This tool provides a simple interface for managing inventory in real time. It focuses on speed, simplicity, and live collaboration rather than complex enterprise features.

Core idea:
- Keep inventory operations fast and minimal
- Allow multiple users to update stock simultaneously
- Reflect changes instantly across all clients

---

## Features

- Real-time stock table updates
- Add, edit, and delete inventory items
- Inline quantity adjustment (+ / - controls)
- Modal-based input system
- Authentication (Supabase)
- Row-level security (RLS) for data access control
- Live synchronization via database subscriptions

---

## Tech Stack

- Vanilla JavaScript (ES Modules)
- Supabase (Auth + Database + Realtime)
- HTML + CSS (no framework)
- ESM CDN imports

---

## Project Status

This project is currently under active development, so:
